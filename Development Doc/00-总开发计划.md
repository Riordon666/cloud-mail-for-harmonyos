# 00 - Cloud Mail 鸿蒙移植 · 总开发计划 (修订版)

> **目标**：将 Cloud Mail (Vue 3 + CF Workers) 完整移植为 HarmonyOS (ArkUI + ArkTS) 原生邮件客户端
> **策略**：先画完所有页面 UI（Mock 数据），再接真实 API，最后打磨完善
> **修订原因**：原计划由 AI 编写未参考官方文档，本版基于 Spatialization 案例、HDS 规范修正

---

## 阶段总览

| 阶段 | 步骤 | 目标 |
|------|------|------|
| **一、基础架构** | 01 | UI 框架 + 沉浸光感 + 设计系统（已完成） |
| **二、核心邮件** | 02-04 | 收件箱 → 详情/写邮件 → 星标/发件箱/草稿箱 |
| **三、用户体系** | 05-07 | 登录注册 → 账号管理 → 个人设置 |

| **六、打磨发布** | 13-14 | 主题国际化 → 平板适配/性能优化 |
