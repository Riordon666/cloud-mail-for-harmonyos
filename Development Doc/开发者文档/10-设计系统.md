## HdsNavigation 沉浸光感框架

当前项目使用的核心设计框架，来自 `@kit.UIDesignKit`：

```typescript
import { hdsMaterial, HdsNavigation, HdsTabs,
         HdsTabsController, ScrollEffectType } from '@kit.UIDesignKit'
```

**Material 系统参数**（来自 UI Design Kit 文档）：

| 枚举 | 值 | 效果 |
|------|-----|------|
| `MaterialType.ADAPTIVE` | 自适应 | 跟随系统壁纸颜色自动调整 |
| `MaterialType.IMMERSIVE` | 沉浸式 | 更强模糊 + 光感映射 |
| `MaterialLevel.ADAPTIVE` | 自适应 | 自动选择最合适的材质等级 |
| `MaterialLevel.EXQUISITE` | 精致 | 最强模糊效果 |
| `MaterialLevel.GENTLE` | 柔和 | 中等模糊 |
| `MaterialLevel.SMOOTH` | 平滑 | 轻微模糊 |

**ScrollEffect 类型**：

| 类型 | 效果 |
|------|------|
| `IMMERSIVE_GRADIENT_BLUR` | 滚动时导航栏渐变毛玻璃 ✅ 当前使用 |
| `TRANSPARENT_TO_BLUR` | 透明→模糊渐变 |

## 主题切换实现

```typescript
// ThemeStore.ets
export class ThemeStore {
  static getMode(): number {
    return AppStorage.get<number>('colorMode') ?? 0
  }

  static isDark(): boolean {
    return this.getMode() === 1
  }

  static setMode(isDark: boolean) {
    const mode = isDark ? 1 : 0
    AppStorage.setOrCreate('colorMode', mode)
    // 同步到应用设置
    const context = getContext()
    context.getApplicationContext().setColorMode(
      isDark
        ? ConfigurationConstant.ColorMode.COLOR_MODE_DARK
        : ConfigurationConstant.ColorMode.COLOR_MODE_LIGHT
    )
  }

  static toggle() {
    this.setMode(!this.isDark())
  }
}

// 组件中使用
@Component
struct MyPage {
  @StorageProp('colorMode') colorMode: number = 0

  private getColors() {
    return this.colorMode === 1
      ? { bg: '#0D0D0D', card: '#1A1A1A', text: '#E8E8E8' }
      : { bg: '#F1F3F5', card: '#FFFFFF', text: '#1A1A1A' }
  }
}
```

## 设计理念总结（来自华为官方）

> "One Harmonious Universe" — 一生万物，万物归一

1. **One（归一）**：以人为中心，基于实证设计
2. **Harmonious（和谐）**：物理世界与数字世界的平衡，光/影/材质的映射
3. **Universe（万境）**：多设备自适应，一致而有差异

**对 Cloud Mail 的启示**：
- 保持 iOS/Android 用户熟悉的交互模式
- 利用 HarmonyOS 独家能力（沉浸光感、Material 自适应）
- 平板/折叠屏要考虑多窗口和布局自适应
