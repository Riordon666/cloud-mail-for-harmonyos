# 09 — 程序框架

> Ability Kit 应用生命周期 + 进程线程模型

## UIAbility 生命周期

```
onCreate() → onWindowStageCreate() → onForeground()
                                       ↕ (前后台切换)
onDestroy() ← onWindowStageDestroy() ← onBackground()
```

```typescript
import { AbilityConstant, UIAbility, Want } from '@kit.AbilityKit'
import { window } from '@kit.ArkUI'

export default class MailAbility extends UIAbility {
  // 1. 创建时：初始化全局配置
  onCreate(want: Want, launchParam: AbilityConstant.LaunchParam): void {
    // 设置颜色模式（不跟随系统，由应用内切换）
    this.context.getApplicationContext()
      .setColorMode(ConfigurationConstant.ColorMode.COLOR_MODE_NOT_SET)
  }

  // 2. 窗口创建：加载页面
  onWindowStageCreate(windowStage: window.WindowStage): void {
    windowStage.loadContent('pages/Index', (err) => {
      if (err.code) {
        console.error('页面加载失败: ' + JSON.stringify(err))
      }
    })
  }

  // 3. 前台：恢复界面（如网络重连）
  onForeground(): void {
    // 恢复定时刷新、重连WebSocket等
  }

  // 4. 后台：保存状态
  onBackground(): void {
    // 保存草稿、暂停轮询
  }

  // 5. 窗口销毁
  onWindowStageDestroy(): void {}

  // 6. 销毁
  onDestroy(): void {}
}
```

## 冷启动 vs 热启动

| 启动类型 | 触发场景 | 性能 |
|---------|---------|------|
| **冷启动** | 首次启动 / 进程被杀死后启动 | 慢（完整初始化） |
| **热启动** | 从后台切回 / 近期启动过 | 快（进程仍在） |

```typescript
onCreate(want: Want, launchParam: AbilityConstant.LaunchParam): void {
  if (launchParam.launchReason === AbilityConstant.LaunchReason.START_ABILITY) {
    // 冷启动 → 初始化数据库、创建目录
  } else if (launchParam.launchReason
    === AbilityConstant.LaunchReason.CONTINUATION) {
    // 从其他设备流转
  }
}
```

## 进程线程模型

| 类型 | 用途 | 限制 |
|------|------|------|
| **主线程** | UI 渲染 + ArkTS 引擎 | ❌ 不能阻塞（不要同步网络请求） |
| **TaskPool** | 短时后台任务（< 3 分钟） | 独立 JS 环境，不共享对象 |
| **Worker** | 长时后台任务 | 独立 JS 环境，消息通信 |

### TaskPool — 邮件同步示例

```typescript
import { taskpool } from '@kit.ArkTS'

// 后台同步邮件
@Concurrent
async function syncEmailsTask(token: string, accountId: number): Promise<EmailData[]> {
  // TaskPool 中不能直接访问 UI 状态
  // 通过 HTTP 获取数据，返回结果
  const response = await fetch(`https://riordon.xyz/api/email/latest?...`)
  return response.json()
}

// 主线程调用
async function triggerSync() {
  try {
    const task = new taskpool.Task(syncEmailsTask,
      AppStorage.get('token'), this.currentAccountId)
    const newEmails = await taskpool.execute(task) as EmailData[]
    // 回到主线程更新 UI
    this.emails = [...newEmails, ...this.emails]
  } catch (err) {
    console.error('同步失败:', err)
  }
}
```

### Worker — 长时轮询

```typescript
// worker/mailPoller.ts (Worker 线程)
import { worker } from '@kit.ArkTS'

const parentPort = worker.workerPort

// 每 30 秒检查新邮件
setInterval(async () => {
  const response = await fetch('https://riordon.xyz/api/email/latest?...')
  const newEmails = await response.json()
  parentPort.postMessage({ type: 'newEmails', data: newEmails })
}, 30000)
```

## 后台任务

### 定时任务（每小时同步一次邮件）

```typescript
import { backgroundTaskManager } from '@kit.BackgroundTasksKit'

// 申请长时任务
async function requestContinuousTask() {
  const wantAgentInfo: backgroundTaskManager.WantAgentInfo = {
    wants: [{ bundleName: 'com.example.cloudmail', abilityName: 'MailAbility' }],
    operationType: backgroundTaskManager.OperationType.START_ABILITY,
    requestCode: 0,
    wantAgentFlags: [backgroundTaskManager.WantAgentFlags.UPDATE_PRESENT_FLAG]
  }
  const wantAgent = await backgroundTaskManager.WantAgent
    .getWantAgent(wantAgentInfo)

  await backgroundTaskManager.startBackgroundRunning(
    this.context,
    backgroundTaskManager.BackgroundMode.DATA_TRANSFER,  // 数据传输
    wantAgent
  )
}
```

## Context 属性表

| 属性 | 路径 | 用途 |
|------|------|------|
| `context.filesDir` | `/data/.../files` | 持久文件 |
| `context.cacheDir` | `/data/.../cache` | 缓存 |
| `context.tempDir` | `/data/.../temp` | 临时文件 |
| `context.databaseDir` | `/data/.../database` | SQLite |
| `context.preferencesDir` | `/data/.../preferences` | Preferences |
| `context.distributedFilesDir` | `/data/.../distributedfiles` | 跨设备文件 |
| `context.bundleCodeDir` | `/data/.../haps/entry` | 应用包路径（只读） |
