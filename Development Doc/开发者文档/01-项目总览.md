# 01 — 项目总览

## 原项目 vs HarmonyOS 架构对比

```
原项目 (Cloud Mail)              HarmonyOS 移植版
──────────────────────────────────────────────────
Vue 3 + Vite                  → ArkUI (ArkTS 声明式)
Element Plus                  → @kit.UIDesignKit + 系统组件
Vue Router                    → Navigation + NavPathStack
Pinia (10 stores)             → @State + AppStorage + PersistentStorage
axios (15 request modules)    → @kit.NetworkKit (HTTP)
Drizzle ORM + D1              → RelationalStore (SQLite)
Cloudflare KV                 → Preferences / KV-Store
Cloudflare R2                 → Core File Kit + 云存储
Cloudflare Email              → @kit.NetworkKit (HTTP send)
Cloudflare Workers AI         → @kit.IntelligenceKit
TinyMCE                       → RichEditor / TextArea (简化)
PostalMime (邮件解析)          → 后端API处理（客户端不解析MIME）
```
