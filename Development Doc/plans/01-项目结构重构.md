# 01 — 项目结构重构

> **前置条件**：无
> **目标**：拆分 `Index.ets`（276 行单文件）→ 独立页面文件 + 可复用组件
> **状态**：✅ 已完成 → [F01 总结](F01-项目结构重构.md)

## 当前问题

`Index.ets` 包含 7 个 `@Component`（Index + InboxView + StarredView + ComposeView + AccountsView + MeView + EmailCard + AccountCard），全部在一个文件中。难以维护、无法子页面导航。

## 目标结构

```
mail/src/main/ets/
├── common/
│   ├── Constants.ets          ✅ 已有（不动）
│   └── MockData.ets           ⏳ 新建（集中管理 mock 数据）
├── components/
│   ├── EmailCard.ets          ⏳ 从 Index.ets 提取
│   ├── AccountCard.ets        ⏳ 从 Index.ets 提取
│   └── AvatarBar.ets          ⏳ 新建（头像 + 首字母）

│   ├── Index.ets              🔧 重构：HdsNavigation 框架 + 悬浮写邮件球

│   ├── SentPage.ets           ⏳ 已发送
```
