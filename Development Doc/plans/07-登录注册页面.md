# 07 — 登录 + 注册页面

> **前置条件**：02 完成
> **对标原项目**：`views/login/index.vue`

## 任务清单

### 任务 1：页面设计
- [ ] 全屏背景（渐变：蓝→紫→粉，对标原项目登录背景）
- [ ] 毛玻璃卡片居中（backdropBlur + 半透明白底）
- [ ] Logo 区域（应用名 "Cloud Mail" + 图标）
- [ ] 登录/注册切换按钮（顶部两个 Tab）

### 任务 2：登录表单
- [ ] 邮箱输入框（InputType.Email，envelope 图标前缀）
- [ ] 密码输入框（InputType.Password，lock 图标前缀 + 显隐切换）
- [ ] 登录按钮（全宽、primary 色、loading 状态）
- [ ] 表单校验：邮箱格式、密码不为空
- [ ] 登录成功：保存 Token → 跳转主页
- [ ] 登录失败：Toast 错误消息

### 任务 3：注册表单
- [ ] 邮箱 + 密码 + 确认密码
- [ ] 注册码输入（可选，根据服务器配置）
- [ ] 密码强度提示（弱/中/强）
- [ ] 确认密码一致性校验
- [ ] 注册成功：自动登录 → 跳转主页

### 任务 4：登录态管理
- [ ] 路由守卫：无 Token 时自动跳转登录页
- [ ] 在 Index.ets 的 aboutToAppear 检查 Token
- [ ] Token 存入 AppStorage + Preferences（双重保险）

### 任务 5：登录背景设置
- [ ] 支持自定义背景图（从 API 获取 setting.background）
- [ ] 背景图加载失败 → 使用默认渐变
- [ ] 背景透明度设置（对标原项目 loginOpacity）

## 验收标准
- ✅ 登录/注册流畅切换
- ✅ 表单校验正常
- ✅ 无 Token → 强制跳转登录
- ✅ 登录成功 → 跳转主页
- ✅ 美观的毛玻璃登录卡片
